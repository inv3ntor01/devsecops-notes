## Completion Workflow

When you complete a module:

1. **Mark it as complete** in this file — change `[ ]` to `[x]`
2. **Add completion date & notes** — what you learned, key takeaways
3. **Commit to GitHub** — include both the module work and this updated tracker
4. **Message format:** `Module X complete: [Module Name] — [Key takeaway]`

### Example Commit:
```
Module 1 complete: Security Gates — Learned about branch protection and pre-commit hooks
```
